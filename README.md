# NomadCode
NomadCode zoom clone coding

## Requirements
1. NodeJs
2. npm i nodemon -D
3. npm i @babel/core @babel/cli @babel/node @babel/preset-env -D
4. npm i express
5. npm i pug
6. npm i ws
7. npm i socket.io
8. npm i @socket.io/admin-ui
    관리자 페이지
9. npm i localtunnel -g
    외부에서 접속 가능(lt --port 3000)
